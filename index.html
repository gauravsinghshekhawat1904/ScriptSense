Handwritten Text Recognition (HTR)

Active

Function

This engine is engineered to solve the "Cursive Problem." It isolates text lines and analyzes pixel density to reconstruct words. A Prefix Tree dictionary runs in the background to correct spelling on-the-fly.

Protocol

  • Upload images of handwritten notes or letters.
  • Adjust Scale if text lines are tightly packed.
  • Toggle Use Dictionary for automated correction.

Structured Invoice Extraction

Active

Function

A multi-modal Vision-Language engine. It "sees" the relationship between labels and values, transforming visual data into clean, machine-readable JSON objects.

Protocol

  • Import Invoices, Receipts, or Purchase Orders.
  • Wait for the Neural Layout Scan to complete.
  • Download the resulting JSON or table for processing.

General Document OCR

Active

Function

Our workhorse pipeline for general conversion. It applies OpenCV adaptive thresholding and Gaussian blurs to ensure high-precision text transcription.

Protocol

  • Upload PDFs or images of printed text.
  • Run Bounding Box preview to verify detection.
  • Execute Full OCR and download as a .txt file.
